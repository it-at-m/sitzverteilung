# Sitzverteilung

The `Sitzverteilung` is a new software, developed by it@M Munich.
Made as a supporting tool after elections, its core functionalities will be calculating the seats for Local Elections, defined by law.
Currently, three different calculations are defined: Hare/Niemayer, Sainte-Lague/Schepers and d'Hondt.



### Built With

The project is built with technologies we use in our projects
([reference architecture](https://opensource.muenchen.de/publish.html#refarch)):

* [Spring Boot](https://spring.io/projects/spring-boot)
* [Vue.js](https://vuejs.org/)

## Roadmap

Since we just started, a roadmap is not available yet.
The project is planned to first have the frontend developed, estimated by March 2025.
When the backend is finished, estimated by the end of May, and final tests are successful, the goal to set the application live is the 30th of June, 2025.

See the [open issues](https://github.com/it-at-m/Sitzverteilung/issues) for a full list of proposed features (and known issues).
